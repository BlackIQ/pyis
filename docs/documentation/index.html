# Introduction


First of all, what is PyIs?

PyIs is a simple module for check variables, url and other things in python.


Why should I use PyIs?

By using PyIs, you write less, cleaner and manage your time too.

Without writing new conditions you just use pyis classes!


How can I install PyIs?

You can use pip to install PyIs or clone repository and install in via python file.
