# Using PyPi

You can install PyIs with pip.

pip3 install pyis

Or you can say specefic version too.


# Install from main repository

In this way you can install PyIs with downloading or cloning repository.

git clone https://gihub.com/BlackIQ/pyis && cd pyis

pip3 install . -e
