# Py Is (pyis)

**pyis** is a library to check values in your python projects.

Check these out

- Installation
- Documentation

### Installation

- Way number 1 : **Using pip**

```
pip3 install pyis
```

- Way number 2 : **Clone and install**

```
git clone https://github.com/BlackIQ/pyis && cd pyis
pip3 install -e .
```
